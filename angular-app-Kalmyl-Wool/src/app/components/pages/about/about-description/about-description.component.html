Как было озвучено на рабочем совещании, специализированная организация по продаже тонкой и полутонкой шерсти в республике ООО «Калмык Вул» (Kalmyk Wool) будет базироваться в поселке Утта Яшкульского района, где уже имеются 2 специальных склада. Калмыцкая шерсть выйдет на международный рынок и получит свой бренд «Kalmyk Wool».
